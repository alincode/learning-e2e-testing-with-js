DEV

  • 開發環境
  • 資料庫：假資料

QA

  • 有時又稱使用者驗收測試(UAT - User Acceptance Testing)
  • 使用情境：驗收功能

Staging

  • 跟 Production 一致的環境
  • 資料庫：與正式環境一致

Production

  • 正式環境
